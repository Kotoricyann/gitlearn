更改用户名/email
git config --global user.name "FR Lv"

将当前目录变成仓库
git init

添加文件
git add git.txt

提交，并添加描述abc
git commit -m "abc"

仓库状态
git status

查看区别
git diff example.txt

查看日志
git log

回滚操作(^ once; ~n nth; 版本号 该版本)
git reset --hard HEAD^

查看版本修改日志
git reflog

撤销修改//用版本库退回工作区
git checkout -- example.txt

回退暂存区到工作区
git reset HEAD example.txt

删除文件
git rm example.txt

生成ssh
ssh-keygen -t rsa -C "youremail@example.com"

连接远程库
git remote add origin git@github.com:michaelliao/learngit.git

推送到远程库
git push (-u) origin master

从远程库克隆
git clone

创建并切换分支（-b）
git checkout -b dev
= git branch dev
  git checkout dev
  
合并指定分支dev到当前分支
git merge dev

删除分支dev
git branch -d dev

保存现场（工作区）
git stash

查看现场
git stash list

恢复现场
git stash pop
git stash apply 

删除现场
git stash drop

添加标签 
git tag <tagname>
